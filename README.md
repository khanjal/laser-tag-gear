# Laser Tag Gear

Workspace for the Laser Tag Gear catalog rebuild.

## Structure
- `app/`: Angular application scaffold (SSR enabled)
- `docs/PROJECT_PLAN.md`: living project plan and milestone tracker
- `Laser Tag Gear.pdf`: design/reference document

## Quick Start
```powershell
cd app
npm start
```
