Overview
Laser Tag type equipment was in use as early as the 70's, but did not make commercial use to early 80's. On this page I list all the known type of Laser Tag equipment. I also have links to specific gear who's popularity has provided me with enough info to bring to you. This page mainly contains home based Laser Tag gear (HBLT) but does include some Arena gear.
Some of the gear has a PDF file link which further describes the gear officially. Usually these are instruction manuals. If you don't see a link then I don't have a manual for that gear. If you happen to have that manual and the time, please scan it in (jpeg or pdf) and send it to manual@khanjal.com. It is greatly appreciated and I will put up the link as soon as possible. Also, to read the manuals online, you'll need Adobe Acrobat Reader.

Laser Challenge top
Manufacturer- Toymax
Laser Challenge is by far the most popular of the new age Laser Tag gear. It has several subset versions, each compatible to a degree.
  • Laser Challenge Gear

Laser Command/Laser Attack top
Manufacturer- Astronomical Toys
This is a short lived Laser Tag system that was made in the late 90's. It was fully compatible with Laser Challenge, which helped it grow somewhat popular. I myself only have the rifle from this gear. They also seem to have released Laser Attack, which is identical to Laser Command.
  • Laser Command Gear

Lazer Tag top
Manufacturer- Tiger Toys and Worlds of Wonder
This has to be the main competitor with Laser Challenge. This gear was originally made in the early 80's, but Tiger Toys reused the name for their current gear.

Quick Shot top
Manufacturer- Radio Shack
This is a one time release gear. Not bad from what I was able to see.
  • Quick Shot

SEGA Lock-On top
Manufacturer- SEGA
This Laser Tag system has a lot of features, including a hit counter, and a lock on beam.
  • SEGA Lock-On

Voice Command Lock-On top
Manufacturer- Playmates Toys
This is like the SEGA Lock-On gear, but has a voice command feature to change modes.
  • Voice Command Lock-On
