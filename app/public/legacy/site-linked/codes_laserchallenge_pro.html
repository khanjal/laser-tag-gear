Overview
The Laser Challenge Pro vests were the first Laser Challenge gear to feature code inputs on the gun to activate special abilities. 1 digit and 2 digit codes are used to enter special game modes, while the 3 digit ones are used to activate special abilities during any game mode. The only buttons on the gun are 1, 2, 3, and e. You usually enter a given code and hit e to complete it. If you enter more then 3 digits it just takes the last 3 you entered and uses them.
Below is a brief list of all the codes, but Fath0m's page has a more detailed list of them, and tips to extend some features.
