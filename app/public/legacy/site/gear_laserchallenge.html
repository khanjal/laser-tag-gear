Overview
Laser Challenge is currently the most popular laser tag brand. Some laser tag gear produced by them has been questionable, but there have been some great ones. Each "series" listed below has differences in it that sets them apart. Ironically, the Original category will probably contain the oldest, and the newest gear.

Laser Challenge Original top
Released- 1996-2003
The first of the LC line, the original gear is the second most used, due to it's cheapness and to the fact it can be more easily found then all the other kinds. On this page, I have all the oddball gear Toymax came out with including remakes of the old gear.
  • Laser Challenge Original and Other Gear

Laser Challenge Pro top
Released- 1997-1998
Toymax tried an attempt at making commercial arena gear. At first it worked, but all of mine, within two months started failing. I have done all the repairs I could, but there are still some occasional problems. Considering I bought this gear fresh of the shelf when it came out ($120 for a twin pack) it was not worth the money, and even for $10 a twin pack wouldn't be. This gear shouldn't be invested in.
  • Laser Challenge Pro Gear

Laser Challenge V2 top
Released- 1998
This by far has to be the best "series" made by Toymax. I currently use this equipment for my league, and have found it to be quite reliable. Except for some small problems, this gear handles all sorts of weather/conditions. Alas, it is no longer produced, and can be hard to come by, most of the time being sold for more then it originally sold for. I knew I should have stocked up when TRU put it on clearance.
  • Laser Challenge V2 Gear

Laser Challenge Extreme top
Released- 1999
This category includes both Radar Extreme and Gotcha Extreme. While Radar is much better then Gotcha, they both include the same features which make them like brother and sister (the brother being the better one ;) ). This gear uses SEGA Lock-On type technology.
