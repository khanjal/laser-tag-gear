Overview
Luckily I have a friend who has a set of this. I have gathered all the information I could from it. He received it in 1990 as a present, and several pieces were still missing from it. Although I received information that this gear is made by Radio Shack, I found no proof on the gun itself.

Gear top
All the gear is aquired in buying one set.

Lock-On top
Released- 1990
Battery Req.: 4 AA's
Set(s): Quick Shot Set
Info: This allows the sensor to be plugged into it and it can receive unlimited hits. When it is hit it has a falling bomb effect. It can be strapped to your body or cliped. Looks like you would mainly use this for target practice.

Lazer Pistol top
Released- 1990
Battery Req.: 4 AA's
Range: 100 ft.
Ammo: Unlimited
Set(s): Quick Shot Set
Info: The pistol comes with a scope and a shoulder rest. The sensor can also be attached to the pistol, probably adding limited hits to it. I was unable to test this with another due to having only one set at my disposal.

Sensor top
Released- 1990
Battery Req.: --None--
Set(s): Quick Shot Set
Info: The sensor plugs into either the Electronic Target or Lazer Pistol, allowing either to be shot.

Sets top
Quick Shot was only available in one set.

Quick Shot Set top
Released- 1990
Model Number: QS2040
Battery Req.: 8 AA's
Contents: 1 Lazer Pistol, 1 Electronic Target, and 1 Sensor
Original Price: $?
Info: This set is the only set that I know for this gear. I wish I had access to two to test them out further.
